# backend_house_price_prediction_python
backend of house price prediction on python

How to deploy?
1. Push backend to a new GitHub repo

2. Create requirements.txt:

fastapi
uvicorn
python-multipart
xgboost
lightgbm
scikit-learn
pandas
numpy

3. Go to railway.app → New Project → Deploy from GitHub repo. Same requirements.txt as above. Railway auto-detects FastAPI and sets everything up.

4. After deploying — update React app
Open HousePricePredictor.jsx and change line 3 from:
const API = "http://localhost:8000";
Replace with deployed URL generated by Railway.

One more thing — CORS is already handled
app.py already has allow_origins=["*"] so GitHub Pages can call it without any browser errors. we don't need to change anything in the backend.
